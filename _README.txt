HOW TO START THIS PROJECT

(1) Clone this Repository on your local machine

(2) Download & Install XAMPP

(3) Create Database in MyPHPAdmin and import file name "hbwebsite"

(4) Import Database and also copy hbwebsite folder to XAMPP/htdocs/

(5) Start SQL AND APACHE server

(6) Open Browser and type "localhost/hbwebsite/" and you are ready to go!
